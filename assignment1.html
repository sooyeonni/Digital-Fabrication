Modeling and Laser Cutting

Assignment 1

                     

Ideation

For the modeling ideation, I was inspired by the hexagonal pattern of the bee house. Using the regular hexagonal patterns, I want to manipulate them into one sizeable 3D figure, so it can show both organized and unorganized patterns when each asset is combined.

Rhino Modeling

For the single asset, I use a hexagon with three stripe apertures. Each aperture can be connected to make different figures. I assembled four assets into one style and combined them to create a large and complex structure.

Laser Cutter Setting

For the laser cutter setting, I set speed as 25%, power as 50%. After testing the size of the two different model, I decided to print the smaller assets.
