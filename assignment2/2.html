Parametric Modelling with Grasshopper

Assignment 2

Extrude

Make the base structure extrude so that it can show the estimated version of the assemble model. To set the extrusion rate, I used number slider to estimate the size and use "extrude" to make its depth.

                     

                     

Grasshoper Definition

For the base rectangle and each tooth of the pieces, I used number slider and unit X,Y,Z to set the size. With the unit Z

Above image explains the grasshoper definition of the extrusion. Use "extrusion" and connect them into each unit vector, so that it can extrude every curbs together.

To make every joints and curve connected, I used "Brep Join" and them merge them into one pieces to cut them into one piece in the laser cutter.

Assembled assets

assembled four assets into one style and combined them to create a large and complex structure.

Laser Cutter Setting

For the laser cutter setting, I set speed as 24%, power as 50%. The asset itself has lot of angles and complex, so I lowered the spead rate to prevent any fire or wrong cut.

Final Deliverable

For the final deliverable, I assembled every pieces into one unit contains two pieces, and then try to connect them look like a parametric wall.
